# AI Disaster Response Assistant

## Problem
Emergency services receive large volumes of disaster reports and cannot prioritize them quickly.

## Solution
Build an AI-powered system that analyzes citizen reports and classifies disasters.

## Features
- Accept disaster reports
- Analyze disaster type
- Assign priority level
- Store reports in database
- Provide real-time disaster insights

## AWS Services
- API Gateway
- Lambda
- DynamoDB
- Amazon Bedrock

## Architecture
User → API Gateway → Lambda → Bedrock → DynamoDB
